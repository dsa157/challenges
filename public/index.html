Challenges API

This is a serverless API for daily challenges. Use the following endpoints:

Search Challenges

GET /api/search?month=jun&day=07

Parameters:

  • month - Required. Month in lowercase (e.g., 'jun')
  • day - Optional. Day of the month (e.g., '07' or '7'). Defaults to current day.

Example: /api/search?month=jun&day=07

Available Months

GET /api/months

Returns a list of available months with challenges.

Example: /api/months

For more information, see the GitHub repository.
